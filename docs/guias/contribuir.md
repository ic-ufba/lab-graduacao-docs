# Guia de Contribuição

Obrigado por considerar contribuir para o nossos projeto!
Este guia tem como objetivo facilitar o processo de contribuição e garantir que todos os colaboradores sigam as mesmas diretrizes.
Se você tiver alguma dúvida ou precisar de ajuda, sinta-se à vontade para entrar em contato através das issues ou do nosso canal de comunicação.

## Como Contribuir

1. **Reportar Bugs**
   - Verifique se o bug já foi reportado na seção issues do projeto da qual deseja contribuir.
   - Se o bug ainda não foi reportado, crie uma nova issue com uma descrição clara e detalhada do problema.

2. **Sugerir Melhorias**
   - Navegue pelas issues existentes para ver se sua sugestão já foi discutida.
   - Caso contrário, crie uma nova issue descrevendo sua sugestão de melhoria.

3. **Enviar Pull Requests**
   - Faça um fork do repositório e crie uma nova branch para sua contribuição.
   - Certifique-se de que seu código segue as diretrizes de estilo e padrões do projeto.
   - Envie um pull request com uma descrição clara das mudanças propostas.
